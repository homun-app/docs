# Build from Source

Build Homun directly from source for maximum control and performance.

## Prerequisites

<Callout type="warning">
Homun requires **Rust 1.75+**. Check your version with `rustc --version`. If you need to update, run `rustup update`.
</Callout>

- **Rust 1.75+** -- install via [rustup](https://rustup.rs/)
- **System dependencies** (varies by OS):

<Tabs items={['macOS', 'Ubuntu / Debian', 'Fedora', 'Arch Linux']}>
  <Tabs.Tab>
```bash
xcode-select --install
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
sudo apt install build-essential pkg-config libssl-dev libsqlite3-dev
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
sudo dnf install gcc openssl-devel sqlite-devel
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
sudo pacman -S base-devel openssl sqlite
```
  </Tabs.Tab>
</Tabs>

## Build

<Steps>
### Clone the repository

```bash
git clone https://github.com/homunbot/homunbot.git
cd homunbot
```

### Build the release binary

```bash
cargo build --release
```

The binary will be at `target/release/homun`.

### Optional: install globally

```bash
cargo install --path .
```

This places `homun` in `~/.cargo/bin/`, which should be in your `PATH`.
</Steps>

## First Run

<Steps>
### Initialize configuration

```bash
homun config
```

This creates `~/.homun/config.toml` with default settings. Edit it to add your LLM provider:

```toml
[provider]
default_model = "anthropic/claude-sonnet-4-20250514"

[provider.anthropic]
api_key = "sk-ant-..."
```

### Start the gateway

```bash
homun gateway
```

This starts the Web UI, all configured channels, scheduled tasks, and the heartbeat system.

### Open the Web UI

Navigate to [http://localhost:18080](http://localhost:18080) to complete setup.
</Steps>

## File Locations

| Path | Purpose |
|---|---|
| `~/.homun/config.toml` | Configuration file |
| `~/.homun/homun.db` | SQLite database |
| `~/.homun/secrets.enc` | Encrypted vault |
| `~/.homun/brain/` | Agent memory (USER.md, SOUL.md) |
| `~/.homun/skills/` | Installed skills |

## Quick Chat (no gateway)

For a quick interactive session without starting the full gateway:

```bash
homun chat
```

Or send a one-shot message:

```bash
homun chat -m "What's the weather in Rome?"
```

## Verbose Logging

<Callout type="info">
Use `RUST_LOG=debug` to see detailed logs for troubleshooting. You can filter by module: `RUST_LOG=homun=debug,sqlx=warn`.
</Callout>

```bash
RUST_LOG=debug homun gateway
```
